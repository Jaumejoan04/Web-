EJERCICIO SELECTORES
A. AGRUPACIÓN DE SELECTORES:
EJERCICIO 1 EXPLICACIÓN:

En este ejercicio vamos a utilizar el selector múltiple, su sintaxis es esta: "elemento, elemento {atributos}" y sirve para seleccionar varios elementos a la vez.

EJERCICIO 1 MUESTREO:

Los dos headings tienen el fondo naranja ya que están los dos en el selector

Los dos headings tienen el fondo naranja ya que están los dos en el selector

EJERCICIO 1 CÓDIGO REAL:

<style>
            h1, h2 {
            background-color: orange;
        }
            </style>
            <body>
                 <h1>Los dos headings tienen el fondo naranja ya que están los dos seleccionados en el selector</h1>
                 <h2>Los dos headings tienen el fondo naranja ya que están los dos seleccionados en el selector</h2>
                 </body>


EJERCICIO 2 EXPLICACIÓN:

En este ejercicio vamos a utilizar un selector más concreto, su sintaxis es esta: "elemento elemento {atributos}", la sintaxis de este selector es casi idéntica a la anterior pero quitando la coma entre los elementos. Este selector sirve para especificar un elemento dentro de otro, como por ejemplo un párrafo dentro de un div "div p {atributos}".

EJERCICIO 8 EXPLICACIÓN:

En este ejercicio veremos 3 tipos de selectores muy similares entre si. Su sintaxis es: "[elemento/^/*/$/="atributo"]". Estos selectores tienen la misma función pero con diferentes "parámetros", se usan para seleccionar elementos con un atributo específico. El selector que contiene el símbolo "^", se utiliza para seleccionar un elemento o un atributo que EMPIEZE por un determinado valor. El selector que contiene el símbolo "*", se utiliza para seleccionar un elemento que contenga un determinado valor. Y por último el selector que contiene el símbolo "$", se utiliza para seleccionar un elemento que acabe con un determinado valor. En el siguiente ejercicio se puede ver como se puede utilizar con el elemento "class".

EJERCICIO 11 EXPLICACIÓN:

Los 3 selectores que utilizamos aquí son: "first-of-type", "last-of-type" y "only-of-type". Estos 3 selectores, también se parecen mucho pero tienen funciones distintas. El primer selector "first-of-type" sirve para estilizar o dar atributos a un elemento que es el primero de su tipo en su nivel, por ejemplo el primero dentro de un div. El segundo selector "last-of-type" sirve para estilizar o dar atributos a un elemento que es el último de su tipo en su nivel, por ejemplo el último dentro de un div. El tercero "only-of-type" sirve para estilizar o dar atributos a un elemento que és el único de su tipo en su nivel, si hay más de un elemento del mismo tipo, este pseudoelemento no funcionará .
